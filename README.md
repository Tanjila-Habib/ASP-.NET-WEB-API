 📅 Appointment Management System – ASP.NET Core Web API
📌Project Review

A backend-only RESTful API built with ASP.NET Core Web API for managing appointments, designed using a scalable N-Tier architecture.
This project is an Appointment Management System built entirely on the backend. It follows RESTful API principles and uses JSON-based request and response formats along with appropriate HTTP methods to handle client requests.

The project is structured using an N-Tier Architecture to ensure modularity, scalability, and maintainability. It consists of the foowing layers:
1. Presentation Layer (API Controllers) – Handles HTTP requests and responses.
2. Business Logic Layer (BLL) – Implements core application logic.
3. Data Access Layer (DAL) – Manages database interactions.
   
The system uses a Relational Database Management System to store data and follows best practices for code structure, validation, and error handling.
